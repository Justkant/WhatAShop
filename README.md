## Steps to start development:
- Install rethinkdb
- Start with ```rethinkdb --daemon``` or simply ```rethinkdb``` in a new terminal
- Then start the npm script for development with ```npm run dev```

## Preferred IDE:
- WebStorm 11
- SublimeText
- Atom
